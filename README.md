# вњЁ About The Project

**Eco Shopnexa** is a modern, scalable, CMS-driven professional e-commerce web application built using **React.js**, **Tailwind CSS v3**, **Redux Toolkit**, and **TanStack Query**.

The goal of this project was to build a **real-world frontend architecture** that includes:

- Dynamic API-driven routing
- Reusable scalable components
- CMS-ready structure
- Optimized state management
- Production-level UI/UX
- Responsive modern layouts
- Live backend integration

This project simulates how professional production-grade e-commerce systems are structured in modern frontend development.
